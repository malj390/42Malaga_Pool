# Vim 

## Shortcuts

### Insert Mode

```
Ctrl + N - Autocomplete

```
### Escape Mode

```
:q - Exit
:q! - Exit 
:w - write
:wq or :x - write and exit
gg - Go to the beggining of a document
G - go to the end

```
## Settings

### Adviced to have in .vimrc

```
set number
set mouse=a

```

### Others

```
:set list
:set invnumber
:set autoindent (or set ai)

```
